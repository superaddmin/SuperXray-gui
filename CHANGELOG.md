# CHANGELOG

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 与语义化版本号。

## [2.9.9] - 2026-04-30

### Added

- 增加入站创建表单双语字段提示与字段级帮助弹窗，覆盖基础信息、客户端、TLS/Reality 与常见传输字段。
- 增加入站表单帮助与 TLS/Reality 前端校验的 Node 回归测试。

### Changed

- 优化入站创建弹窗宽度、滚动区域、客户面板默认展开和 Flow 字段可发现性。
- 将项目版本号更新为 `2.9.9`，并同步部署文档、README 与技术文档中的版本引用。

### Fixed

- 在 Reality 私钥为空、TLS 证书字段为空时提前给出中英双语前端拦截提示，减少提交后才由后端或 Xray 返回错误的情况。

## [2.9.8] - 2026-04-30

### Added

- 增加入站规则创建填写教程，补充常见协议、传输、TLS/Reality 与客户端字段的填写说明。
- 增加入站前端、后端 JSON 解析、cron 防重入和 Xray API 同步锁的回归测试。

### Fixed

- 修复入站列表添加/编辑客户端时前端 JSON 拼接不稳定、路径参数未编码和变量泄漏风险。
- 修复后端入站客户端 JSON 入口在 `settings.clients` 缺失或类型错误时可能 panic 的问题。
- 修复定时任务和前端操作同时同步 Xray API 时可能发生重入和调用交错的问题。

### Changed

- 优化入站创建弹窗字段排列，将基础信息、连接方式、安全层和协议客户端字段调整为更符合填写流程的顺序。
- 将项目版本号更新为 `2.9.8`，并同步文档中的版本引用。

### Changed

## [2.9.4] - 2026-04-29

### Added

- 增加 Ubuntu 服务器部署说明，覆盖 amd64 与 arm64 架构、Release 包安装、源码构建、证书配置和常见故障排查。
- 增加开发文档、模块说明和多语言 README/翻译配置校验说明。

### Changed

- 将服务器安装脚本和面板管理脚本的高频交互提示中文化，覆盖首次安装、证书配置、服务启停、Fail2ban/IP 限制和常见状态提示。

### Fixed

- 修复 Release 资源命名与安装脚本下载路径不一致导致的服务器安装失败风险。
- 修复路径根封装、错误处理、配置 JSON 拼接和前端日志显示相关安全问题。
- 修复 ARM64 构建与验证流程中的 CGO 编译环境约束。

### Security

- 强化 Web 服务安全基线，包括安全响应头、可信代理、Host 校验、限流和敏感配置处理。
- 增加 DOM XSS 回归验证和 Go race 测试验证路径。
